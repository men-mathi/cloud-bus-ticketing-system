Smart Bus Connect is a cloud-based public transport ticketing platform designed to simplify bus travel across cities.

The system allows passengers to search for available buses, select seats visually, make secure online payments, and receive QR-code based digital tickets instantly.

This project demonstrates a modern ticket booking workflow including seat layout management, payment simulation, QR ticket generation, and an admin dashboard for bus management.

Key Features:
• Online bus booking
• Interactive seat selection
• Secure payment simulation
• QR-code ticket generation
• Admin dashboard
• Responsive UI design
• Cloud deployment ready architecture

This project was developed as a real-world system design demonstration for modern public transport ticketing solutions.
